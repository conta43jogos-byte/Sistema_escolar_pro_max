# 🚀 GUIA RÁPIDO - Como Rodar o App

## ⚡ 5 Passos Rápidos

### 1️⃣ Abra o Terminal/PowerShell
Na pasta do projeto (Sistema_escolar_pro_max)

### 2️⃣ Instale as Dependências
```bash
npm install
```
⏱️ Espere ~2-5 minutos na primeira vez

### 3️⃣ Inicie o Servidor
```bash
npm run dev
```

### 4️⃣ Abra no Navegador
```
http://localhost:5173
```

### 5️⃣ Faça Login!
- **Email**: `admin@example.com` (ou qualquer email)
- **Senha**: `123456` (ou qualquer senha)

## ❓ Problemas Comuns

| Problema | Solução |
|----------|---------|
| "Port 5173 in use" | Vite usa próxima porta automaticamente |
| "npm not found" | Instale Node.js: https://nodejs.org |
| Página em branco | Limpe cache: Ctrl+Shift+Del |
| Erro de import | Execute `npm install` novamente |
